# BreadTracker

## Overview

BreadTracker is an Android budgeting and expense tracking application developed using Java, Android Studio, and SQLite. The app helps users record expenses, manage spending categories, and monitor monthly budget goals in a simple and user-friendly way.

## Target Users

BreadTracker is designed for:

* Students managing personal finances
* Young professionals tracking monthly spending
* Anyone looking for a simple budgeting solution

## Features

* User Registration and Login
* Expense Management
* Expense Categories
* Monthly Budget Goals
* Reports and Spending Information
* Budget Progress Tracking

## Design Decisions

### SQLite Database

SQLite was used to store all user and expense data locally, allowing the application to function without an internet connection.

### Dashboard Layout

A dashboard-style home screen was implemented to provide easy navigation between all major features.

### Visual Budget Tracking

A progress bar was used to visually display spending progress against monthly budget goals, making it easier for users to monitor their budget status.

## Custom Features

### 1. Dark Mode

Users can switch between light and dark themes using a toggle switch. The selected theme is saved and automatically applied when the app is reopened.

### 2. Achievement System

An achievement system was added to encourage user engagement. Users unlock badges as they continue adding expenses and using the application.

## GitHub Usage

GitHub was used for version control and project management throughout development. It provided a secure backup of the project, tracked code changes, and allowed the application source code to be stored and managed efficiently.

## Technologies Used

* Java
* Android Studio
* SQLite
* XML
* Git
* GitHub

## Conclusion

BreadTracker provides users with a simple and effective way to manage their finances. By combining expense tracking, budget monitoring, achievements, and dark mode support, the application offers a practical solution for personal budgeting.
